Types de Variables

En Python, les types de données sont : int (entier), float (nombre décimal), str (chaîne de caractères), et bool (booléen). 

nombre = 5  # int pour les nombres simples
prix = 19.99  # float pour les nombres à virgule
nom = "Python"  # string pour les textes
is_active = True  # boolean (vrai ou faux)

Opérateurs

Voici quelques opérateurs courants :

  • + : addition
  • - : soustraction
  • * : multiplication
  • / : division
resultat = 5 + 3  # 8

Les Chaînes de Caractères

Les chaînes de caractères peuvent être manipulées facilement :

nom = "Python"
greeting = "Bonjour, " + nom  # Bonjour, Python
